# 🏦 Investment Agent Team

RS Investment 데이터를 활용해 **AI 투자 팀**이 종목을 분석·추천하고, $100,000 모의 포트폴리오를 운용하며, 싸이월드 미니룸에서 실시간으로 일하는 모습을 보여주는 시스템입니다.

> ⚠️ **모의 투자입니다.** 실제 돈이 오가지 않으며, 교육·훈련 목적입니다. 투자 조언이 아닙니다.

## 🚀 시작하기

클로드에게 두 명령 중 하나를 말하면 됩니다 (자세한 설명: [docs/investment-commands.md](docs/investment-commands.md)):

- **`start investment light`** — 스크리닝 + 시황 심층 + 종목별 상세 리포트 (~5분)
- **`start investment heavy`** — light + 27명 AI 검증단(Opus·Sonnet·Haiku) 심의 (~6~8분)

터미널로 계산기 부분만 돌리려면:

```bash
npm run start-investment
```

이 명령 하나로 아래가 순서대로 실행됩니다:

1. **Alex(분석가)** — 최신 활성 전략(`strategy/active-strategy.md`)으로 RS 데이터 스크리닝
2. **Sara(리스크관리)** — 포지션 사이징, 손절/트레일링 세팅
3. **Jordan(트레이더)** — 모의 거래 기록, 보유분 트레일링 갱신
4. **Morgan(전략가)** — 매매일지 검토, 주기 도래 시 팀 회의 → 전략 진화
5. **미니룸 오픈** — 브라우저에서 미니미들이 한글로 대화하며 일하는 모습 관람
6. **CEO RICH** — Telegram으로 추천종목·누적수익률 보고
7. **GitHub 커밋** — 모든 기록 저장 + Pages 배포

## 📊 전략

- **Perfect Storm** (5단계): 신호등 → 주도섹터 → 진입조건 → 진입신호(60D_BRK+VOL_X+CLS_POS) → 단계적 익절
- **응봉아재보소** (4단계): 시장판단 → 섹터 자금흐름 → 섹터 리더 → 종목 선정
- **살아있는 전략**: 팀 회의로 계속 진화 (`strategy/` 버전 관리)

## 💰 익절/손절 (단계적 익절 + 트레일링)

- 손절: -7% 또는 60일선 2일 이탈
- 1차 익절: 3~5일 급등 시 1/3 매도, 나머지 손절선 본전으로
- 트레일링: 나머지 2/3는 20일선 이탈까지 홀딩

## 🎯 목표
