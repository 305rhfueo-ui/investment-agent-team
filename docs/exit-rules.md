# 💰 익절/손절 규칙 — 단계적 익절 + 트레일링

> "손실은 짧게, 수익은 길게" — 모멘텀(RS) 투자의 정석. 고정 목표가(+50% 일괄 매도)를 쓰지 않는 이유는, 진짜 대박주(NVDA·AMD 등 100~300% 상승)를 조기에 놓치기 때문입니다.

## 손절 (무조건)
1. **-7% 룰**: 종가 기준 진입가 대비 -7% → 강제 청산
2. **60일선 이탈**: 60일 돌파선 아래 2일 연속 종가 → 청산

## 1차 익절
- 진입 후 **3~5일 강한 급등** 시 → **보유량의 1/3 매도**
- 나머지 2/3의 **손절선을 '본전(breakeven)'으로 상향**
- → 이 시점부터 최악의 경우도 손실 0 (리스크 프리 포지션)

## 트레일링 (추세추종)
- 나머지 2/3는 **20일 이동평균 이탈까지 홀딩**
- 추세가 살아있는 한 계속 보유 → 큰 수익 가능

## 장기 홀딩 (선택)
- 초강세 종목은 **월봉 10개월선 이탈까지** 홀딩 (주봉 흔들림 무시)

## 분산 매도 신호
- **50DIV > 70% + 천정 거래량(topping volume)** → 분할 매도 시작
- 큰 음봉 / 거래량 동반 갭 출현 시

## 요약 다이어그램

```
진입
 │
 ├── -7% 도달 ─────────────→ 전량 손절
 │
 ├── +20%(3~5일 급등) ─────→ 1/3 익절, 손절선 본전으로
 │        │
 │        └── 20일선 이탈 ──→ 나머지 2/3 청산 (추세 종료)
 │
 └── 50DIV>70% + 천정거래량 → 분할 매도
```

## 포지션 사이징
```
포지션 크기(주) = (자본 × 2%) ÷ (진입가 × 7%)
예: $100,000 × 2% = $2,000 위험 예산
    진입가 $135.50, 손절 $126.02 (-7%)
    → $2,000 ÷ $9.48 = 약 211주? (아님)
```
> 정확히는: 위험액 $2,000 ÷ 주당 손실($135.50 × 7% = $9.49) = 210주.
> 단, 종목당 최대 투입 비중·총 투입 60% 상한도 함께 적용 (Sara가 조정).
