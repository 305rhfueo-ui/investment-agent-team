# 🚀 투자 실행 명령어 안내 (light / heavy)

투자 분석을 시작하는 명령은 딱 **두 가지**예요. 저(클로드)에게 그대로 말하면 됩니다.

---

## ⚡ `start investment light`

> **"빠르게 오늘 시황이랑 추천 종목 자세히 보고 싶어"** 할 때

**하는 일**
1. 🧮 **실무진(계산기)** — RS 데이터로 종목 스크리닝, 모의매매 기록, Nova 커뮤니티 버즈, 매매일지
2. 🔍 **AI 심층 분석(A)** — 오늘 시황·유동성을 웹검색으로 자세히 + **추천 종목별 상세 분석**
   (회사가 뭐하는 곳인지 → 왜 주목 → 차트 셋업 → 리스크 → 지켜볼 것, 전부 쉬운 말)

**결과물**: 자세하고 쉬운 **투자 브리핑 리포트** (📋 리포트 탭 + Word 저장)

**쓰는 AI**: Opus 1명(시황) + Sonnet 5명(종목) ≈ **6명**
**걸리는 시간**: 약 5분 · **비용**: 보통

---

## ⚖️ `start investment heavy`

> **"이 종목들 진짜 사도 되는지 여러 AI한테 제대로 검증받고 싶어"** 할 때

**하는 일**: `light`가 하는 것 **전부 +** 아래 **27명 AI 검증단(B)**
- 📡 **시장 판단**(Opus) — 지금 신규 진입해도 되는 장인지
- 🔍 **팩트체크**(Haiku) — 함정 종목 거르기(상폐·펌핑·SPAC)
- 💥 **폭발력 패널 3모델** — Opus(펀더멘털)·Sonnet(기술적)·Haiku(촉매/테마)가 각각 점수
- 😈 **악마의 변호인**(Opus) — 사면 안 되는 이유 반박
- ⚖️ **최종 판정**(Opus) — 폭발력 순 3종목 이내(또는 0종목)

**결과물**: `light` 리포트 **+ 검증단 심의**(종목별 점수·거부/관망·최종 추천)
→ 미니룸 **⚖️ 검증단 방**과 📋 리포트에 자동 반영

**쓰는 AI**: **27명** (Opus·Sonnet·Haiku)
**걸리는 시간**: 약 6~8분 · **비용**: 높음

---

## 📊 한눈에 비교

| | ⚡ light | ⚖️ heavy |
|---|---|---|
| 종목 스크리닝·모의매매 | ✅ | ✅ |
| 시황·유동성 심층 | ✅ | ✅ |
| 종목별 상세 분석 | ✅ | ✅ |
| **27명 AI 검증단** | ❌ | ✅ |
| 검증단 방 갱신 | — | ✅ |
| AI 인원 | 약 6명 | 27명 |
| 시간 | ~5분 | ~6~8분 |
| 비용 | 보통 | 높음 |
| 이럴 때 | 매일 빠른 브리핑 | 실제 매수 전 정밀 검증 |

---

## 💡 쉽게 기억하기

- **light = 실무진이 조사해서 자세한 보고서** 📋
- **heavy = 거기에 심의위원단(AI 3모델)이 "진짜 사도 돼?" 최종 검증** ⚖️

## ▶️ 사용법
채팅창에 그냥 이렇게 말하면 됩니다:
```
start investment light
```
```
start investment heavy
```

> 참고: 무인 자동 실행(GitHub Actions, 매일)은 계산기(Node) 부분까지만 돌아가요. AI 심층·검증단은 저(클로드)와 함께 실행할 때 작동합니다(토큰 비용 때문).
