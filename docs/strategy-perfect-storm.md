# ⚡ Perfect Storm 전략 (사이트 원문)

폭발적 상승 초입을 잡는 5단계 프레임워크.

## 1단계 — 시장 신호등
- **진입 허용**: QQQ 종가 > 50일선 (🟢)
- **중단 조건**: 돌파 3회 연속 실패 → 즉시 진입 중단

## 2단계 — 주도 섹터 선정
"찐상승" 필터:
- 최종 WRS 상위 tier
- FRANK 25일 변화 ≥ 0 (순위 개선)
- F25d chg(%) ≥ 0 (점수 상승)
- 보수형 "찐찐상승": F10d chg(%) ≥ 0 추가

## 3단계 — 진입 전 종목 조건

| 지표 | 기준 | 목적 |
|------|------|------|
| 52주 신고가 | ≥ 80% | 가치 함정 제거, 저항 최소화 |
| 이동평균 배열 | 20>60>120 = YES | 정배열(모든 참여자 수익) |
| 50일 이격도 | ≤ 30% (극강세장 40%) | 과열 제거 |
| ADR (일일 변동폭) | ≥ 3.5~4% | 폭발 잠재력 |
| (선택) 최근 강도 | 60일 내 +30% | 폭발 DNA 확인 |

## 4단계 — 진입 신호 (3개 동시 충족)

1. **60D_BRK = YES** — 당일 종가 > 직전 60거래일 최고가(당일 제외)
2. **VOL_X ≥ 2.0** — 거래대금 / 20일 평균 ≥ 200% (기관 확인)
3. **CLS_POS ≥ 50** — 종가가 당일 범위 50% 이상 위치 (윗꼬리 함정 제거)

**실행 규칙**: 다음날 시가 진입. +5% 갭업 추격 금지 (피벗 룰 위반).

## 5단계 — 청산 프로토콜
- **손절**: 종가 기준 -7% 강제 청산 / 60일 돌파선 아래 2일 연속 → 매도
- **1차 익절**: 3~5일 강한 급등 시 1/3 매도, 나머지 2/3 손절선을 본전으로
- **트레일링**: 나머지 2/3는 20일선 이탈까지 홀딩
- **분산 경고**: 50DIV > 70% + 천정 거래량 → 분할 매도 시작

## 보조 지표
- **BBWTHD** ≤ 0.15: 볼린저 밴드 수축(에너지 응축)
- **OBV**: 가격 정체 중 상승 = 숨은 매집
- **MFI**: 80(과매수) → 50 회복 = 매수 가능
- **RS_Dif** 양수 (3M RS > 6M RS): 가속 시작
