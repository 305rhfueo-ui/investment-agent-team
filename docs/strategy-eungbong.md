# 👨‍🏫 응봉아재보소 전략 (사이트 원문)

투자 방법론을 4단계로 정리한 실전 가이드.

## 1단계 — 시장 판단
- QQQ 월봉이 10개월 이동평균 위에 있어야 함 (핵심 조건)
- 급락 후 강한 반등 시 QQQ 일봉 200MA 보조 지표로 활용
- 순현금흐름 모니터링: 지수는 오르는데 유동성 감소 → 폭락 위험 신호

## 2단계 — 섹터 자금흐름 분석
- Sector/Industry WRS 데이터를 WRS 내림차순 정렬
- FRANK 상위 10 섹터 집중, 40~50위도 관찰해 신흥 트렌드 포착
- "찐상승" 필터: FRANK 25일 변화 양수 AND F25d% 변화 양수인 섹터만 선택

## 3단계 — 섹터 리더 식별
- 선택한 섹터 내에서 상대강도 순위 → 상위 1~2 종목이 섹터 리더
- 뉴스 검색으로 시장 인지도 교차 확인

## 4단계 — 종목 선정 & 진입

### 포지션 기준
- **50div(%) ≤ 30%** (극강세장에서만 40% 허용)
- **10div(%) -3% ~ +3%** = 이상적 눌림목 진입
- **중장기 정배열 (20>60>120 MA) = YES** 필수
- 애널리스트 up/down 비율 ≥ 80% 선호

### 실적 기준 (FS 탭)
- 최근 분기(0NI%) > 향후 분기 → 실적 가속 신호
- 향후 EPS 성장률(CY%, NY%) 강세 확인
- 애널리스트 전월 대비 +5% 상향 = 고확률 셋업

## 청산 규칙
- **손실**: -7% 기계적 손절
- **단기 익절**: 큰 음봉, 거래량 동반 갭, 천정 거래량
- **장기 익절**: 월봉 10MA 이탈까지 홀딩 (주봉 흔들림 무시)
